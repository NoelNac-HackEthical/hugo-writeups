## Présentation

Ce dépôt contient les sources du site, développé avec **Hugo** et le thème **PaperMod**.

Le projet regroupe principalement :

- des writeups et walkthroughs Hack The Box en français ;
- des méthodes d’énumération réseau et web ;
- des procédures d’escalade de privilèges Linux ;
- des notes techniques orientées CTF ;
- des workflows reproductibles utilisés en conditions réelles de laboratoire.

L’objectif du site est de proposer des writeups structurés, pédagogiques et exploitables par des utilisateurs souhaitant progresser méthodiquement dans les environnements Capture The Flag (CTF).

Le contenu est actuellement principalement orienté vers les machines **Hack The Box Easy**.

---

## Philosophie du projet

Les writeups et walkthroughs publiés sur le site suivent une approche volontairement :

- progressive ;
- reproductible ;
- documentée ;
- centrée sur l’analyse ;
- fidèle aux étapes réellement effectuées pendant la résolution.

L’objectif n’est pas uniquement de présenter une solution finale, mais de conserver une trace exploitable du raisonnement technique :

- phase d’énumération ;
- validation des hypothèses ;
- exploitation ;
- stabilisation ;
- post-exploitation ;
- escalade de privilèges.

Chaque workflow cherche à rester cohérent avec une méthodologie réaliste utilisée en environnement CTF et en laboratoire.
